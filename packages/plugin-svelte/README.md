# plugin-svelte

## Install

```bash
yarn add @poi/plugin-svelte --dev
```

## Usage

```js
// poi.config.js
module.exports = {
  plugins: [
    require('@poi/plugin-svelte')(/* options */)
  ]
}
```

## API

### options

#### options.loaderOptions

Options for `svelte-loader`.
