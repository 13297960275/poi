# plugin-riot

Use [Riot](http://riotjs.com/) with Poi.

## Install

```bash
yarn add @poi/plugin-riot --dev
```

## Usage

Activate it in config file:

```js
// poi.config.js
module.exports = {
  plugins: [
    require('@poi/plugin-riot')()
  ]
}
```
